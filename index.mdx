`classicborne-server-protocol` is a low-level Node.js library that implements the Minecraft Classic server protocol. It handles TCP connections, WebSocket upgrades for ClassiCube browser clients, CPE negotiation, and packet I/O — so you can focus on building your game logic.

## What it does

The library provides everything needed to accept Minecraft Classic client connections and communicate using the Classic protocol:

- **TCP server** — accepts raw socket connections on any port (default `25565`)
- **WebSocket support** — serves ClassiCube browser clients over WebSocket via `UpgradingHttpServer`
- **CPE negotiation** — automatically negotiates Classic Protocol Extensions with clients that support them
- **13 built-in CPE extensions** — BlockDefinitions, ExtendedPlayerList, LevelEnvironment, and more
- **Code Page 437** — encodes and decodes strings using the CP437 character set required by the protocol

<Note>
This is a **low-level protocol library**, not a full server implementation. It handles networking and protocol encoding but does not provide game mechanics, world storage, or player management. For a batteries-included server, see [classicborne](https://classicborne.bunnynabbit.com/).
</Note>

## Get started

<Steps>
  <Step title="Install the package">
    Install `classicborne-server-protocol` from npm using your preferred package manager.

    <CodeGroup>
    ```bash npm
    npm install classicborne-server-protocol
    ```
    ```bash yarn
    yarn add classicborne-server-protocol
    ```
    ```bash pnpm
    pnpm add classicborne-server-protocol
    ```
    </CodeGroup>
  </Step>
  <Step title="Create a server">
    Import `Server` and create an instance on your chosen port.

    ```js server.mjs
    import { Server } from "classicborne-server-protocol"
    const server = new Server(25565)
    ```
  </Step>
  <Step title="Handle connections">
    Listen for the `clientConnected` event, send server identification, load a level, and configure the spawn point.

    ```js server.mjs
    const bounds = { x: 64, y: 64, z: 64 }
    const levelData = Buffer.alloc(bounds.x * bounds.y * bounds.z)
    const processedLevel = server.utils.processLevel(levelData, bounds.x, bounds.y, bounds.z)

    server.on("clientConnected", async (client, authInfo) => {
      client.serverIdentification("My Server", "Welcome!", 0x00)
      client.loadLevel(await processedLevel, bounds.x, bounds.y, bounds.z, true)
      client.configureSpawn(0, authInfo.username, 0, 0, 0, 0, 0)
    })
    ```
  </Step>
</Steps>
