This guide walks you through creating a working Minecraft Classic server using `classicborne-server-protocol`. By the end, you will have a server that accepts client connections, sends server identification, loads an empty level, and places the player at the spawn point.

<Steps>
  <Step title="Install the package">
    Install `classicborne-server-protocol` from npm. The library has no required peer dependencies beyond Node.js.

    <CodeGroup>
    ```bash npm
    npm install classicborne-server-protocol
    ```
    ```bash yarn
    yarn add classicborne-server-protocol
    ```
    ```bash pnpm
    pnpm add classicborne-server-protocol
    ```
    </CodeGroup>

    Make sure your `package.json` includes `"type": "module"` since the library is ESM-only.
  </Step>

  <Step title="Create a server file">
    Create a file called `server.mjs` (or any `.mjs` file) and import `Server`:

    ```js server.mjs
    import { Server } from "classicborne-server-protocol"

    const server = new Server(25565)
    ```

    The `Server` constructor takes a port number (default `25565`) and an optional host string. It immediately starts listening for TCP connections.
  </Step>

  <Step title="Process a level buffer">
    Before any client connects, prepare your level data. `server.utils.processLevel` takes a raw block buffer and dimensions, prepends the block count as a 32-bit big-endian integer, then gzip-compresses the result. It returns a `Promise<Buffer>`.

    ```js server.mjs
    const bounds = { x: 512, y: 512, z: 512 }

    // Allocate an empty flat world (all air)
    const emptyLevelBuffer = server.utils.processLevel(
      Buffer.alloc(bounds.x * bounds.y * bounds.z),
      bounds.x,
      bounds.y,
      bounds.z
    )
    ```

    Processing the level once outside the connection handler means you only pay the compression cost a single time, no matter how many clients connect.
  </Step>

  <Step title="Handle the clientConnected event">
    Listen for `clientConnected` to respond when a client finishes authenticating. The event provides a `Client` instance and an `authInfo` object.

    ```js server.mjs
    server.on("clientConnected", async (client, authInfo) => {
      // Send server name, MOTD, and user type (0x00 = normal, 0x64 = operator)
      client.serverIdentification("Empty", "So cool!", 0x00)

      // Send the level; pass true as the fifth argument because the buffer is already processed
      client.loadLevel(await emptyLevelBuffer, bounds.x, bounds.y, bounds.z, true)

      // Spawn the player at position (0, 0, 0) with zero yaw and pitch
      client.configureSpawn(0, authInfo.username, 0, 0, 0, 0, 0)
    })
    ```

    The `authInfo` object contains:
    - `authInfo.username` — the player's username as reported by the client
    - `authInfo.key` — the session key used for authentication (compare against the Minecraft heartbeat API if you want to verify identity)
    - `authInfo.extensions` — the list of CPE extensions the client negotiated, if any

    `client.serverIdentification(serverName, motd, userType)` sends the server identification packet. `userType` `0x00` is a normal user; `0x64` grants operator status in the client UI.

    `client.loadLevel(data, x, y, z, processed, callback)` streams the level to the client in 1024-byte chunks. Pass `true` for `processed` when the buffer has already been run through `processLevel`.

    `client.configureSpawn(id, username, x, y, z, yaw, pitch)` sends the spawn player packet. Use `id` `0` and `authInfo.username` for the local player.
  </Step>

  <Step title="Run the server">
    Run your file with Node.js:

    ```bash
    node server.mjs
    ```

    Connect with a Minecraft Classic client (such as ClassiCube) pointing at `localhost:25565`. You should see an empty world and be placed at the origin.
  </Step>
</Steps>

## Complete minimal server

Here is the full working example combining all the steps above, taken directly from `examples/minimum.mjs`:

```js server.mjs
// Bare minimum server operation
import { Server } from "classicborne-server-protocol"
const server = new Server(25565)

const bounds = { x: 512, y: 512, z: 512 }
const emptyLevelBuffer = server.utils.processLevel(
  Buffer.alloc(bounds.x * bounds.y * bounds.z),
  bounds.x,
  bounds.y,
  bounds.z
)

server.on("clientConnected", async (client, authInfo) => {
  client.serverIdentification("Empty", "So cool!", 0x00)
  client.loadLevel(await emptyLevelBuffer, bounds.x, bounds.y, bounds.z, true)
  client.configureSpawn(0, authInfo.username, 0, 0, 0, 0, 0)
})
```
